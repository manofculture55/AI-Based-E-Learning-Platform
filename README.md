# AI Learning Platform

A modern, AI-powered educational platform designed to provide personalized learning experiences through interactive explanations and generated quizzes.

- **Frontend**: React, Vite, CSS Modules (Responsive & Themed).
- **Backend**: Python, Flask, SQLAlchemy.
- **Database**: SQLite (Dev) / PostgreSQL (Prod ready).
- **AI Engine**: Google Gemini Pro API.
- **Authentication**: JWT (JSON Web Tokens).

## Setup & Installation

### Prerequisites
- Node.js (v18+)
- Python (v3.10+)
- Google Gemini API Key

### Backend Setup
1. Navigate to `backend/`:
   ```bash
   cd backend
   ```
2. Create virtual environment:
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Configure environment:
   - Create `.env` file in `backend/`.
   - Add:
     ```env
     # Backend environment variables

     SECRET_KEY=ai-learn-flask-secret-key-2026-prod
     JWT_SECRET_KEY=ai-learn-jwt-super-secret-key-2026-secure

     # üëá PASTE YOUR GEMINI API KEY BELOW (get it from https://aistudio.google.com/apikey)
     GEMINI_API_KEY=KjhgsdfOIHj1cYGjH1udfhER5UJApeibi2y-ABCf
     # this key is not real key it's dummy key
     ```
5. Run server:
   ```bash
   python app.py
   ```

### Frontend Setup
1. Navigate to `frontend/`:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run development server:
   ```bash
   npm run dev
   ```

## Usage

1. **Sign Up/Login**: Create an account to save your history.
2. **Home**: View your dashboard and start a new chat.
3. **Explainer**: Ask for an explanation.
4. **MCQ Generator**: Generate a quiz and test yourself.
5. **History**: Review past explanations and quiz results.

## API Documentation

| Endpoint | Method | Description | Auth |
|---|---|---|---|
| `/api/auth/signup` | POST | Register new user | No |
| `/api/auth/login` | POST | Login user | No |
| `/api/explain` | POST | Generate explanation | Yes |
| `/api/mcq` | POST | Generate MCQs | Yes |
| `/api/mcq/score` | POST | Save quiz score | Yes |
| `/api/history` | GET | Get history | Yes |
| `/api/stats` | GET | Get user stats | Yes |
